# HR Analytics Dashboard – Mini Project

## Project Title

**Workforce Analytics Dashboard: Employee Performance, Salary & Hiring Analysis**

## Project Overview

This project focuses on analyzing employee data using **Microsoft Excel** and **Power BI** to generate meaningful HR insights. The project includes data cleaning, transformation, data modeling, DAX calculations, and interactive dashboard creation to support data-driven HR decision-making.

## Objective

* Clean and prepare raw employee data.
* Analyze employee performance, salary trends, hiring patterns, and workforce distribution.
* Build an interactive Power BI dashboard for HR analysis.
* Provide business insights using descriptive, diagnostic, predictive, and prescriptive analytics.

## Dataset

The project uses two datasets:

### 1. Employee Data

Contains employee information such as:

* Employee ID
* Full Name
* Department
* Job Title
* Gender
* Hire Date
* Termination Date
* Salary
* Manager ID
* Performance Rating
* Employee Status

### 2. Salary Changes

Contains salary revision details such as:

* Change ID
* Employee ID
* Change Date
* Old Salary
* New Salary
* Hike Percentage
* Reason for Salary Change

## Tools Used

* Microsoft Excel
* Power Query
* Microsoft Power BI
* DAX (Data Analysis Expressions)

## Excel Tasks Performed

* Data Cleaning
* Data Formatting
* Text Trimming
* Data Type Conversion
* Missing Value Verification
* Duplicate Data Verification
* Salary Statistics (Minimum, Maximum, Average)
* Pivot Tables
* Salary Status Classification
* Hike Percentage Calculation

## Power BI Tasks Performed

### Data Modelling

* Established a One-to-Many relationship between Employee Data and Salary Changes using EmployeeID.

### DAX Measures

* Total Employees
* Active Employees
* Terminated Employees
* Average Salary
* Average Hike Percentage
* Average Performance Rating

### Dashboard Features

* KPI Cards
* Employee Distribution by Gender
* Employee Status Analysis
* Average Salary by Department
* Employee Count by Department
* Hiring Trend by Year
* Average Performance Rating by Department
* Department Performance Summary
* Year-wise Hiring Summary
* Interactive Slicers and Filters

## Key Insights

* The organization consists of 1,000 employees, with the majority currently active.
* The average salary is approximately ₹92,324.
* The average salary hike is around 9.98%.
* Sales, Logistics, and Quality Control have the highest employee count.
* The Sales department has the highest average salary.
* Average performance ratings across all departments are below 4 out of 5, indicating opportunities for further improvement.
* Salary increments are primarily based on Merit and Promotion.

## Conclusion

This project demonstrates how Excel and Power BI can transform raw HR data into meaningful business insights. The interactive dashboard enables HR professionals to monitor employee performance, salary trends, workforce distribution, and hiring patterns, supporting faster and data-driven decision-making.
